Our Solution


Synaspeak is a software framework designed to detect eye blinks in the form of morse code from a paralysed user and convert it into audible speech.

 

Our Solution


Synaspeak is a software framework designed to detect eye blinks in the form of morse code from a paralysed user and convert it into audible speech.

 

Using a morse code based input aided by auto-prediction algorithms makes synaspeak comparable in performance to state of the art assistive technologies in the market.

User eye blink commands

Blink commands correspond to morse

Input converted to speech output
